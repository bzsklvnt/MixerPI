# CocktailMixer
## Set up project for development
- Create virtual enviroment on Windows:
  -   py -m venv env
  -   .\env\Scripts\activate
  -   pip install -r requirements.txt
- Create virtual enviroment on Linux:
  -   python3 -m venv env
  -   chmod +x ./env/bin/acivate
  -   source ./env/bin/acivate
  -   pip install -r requirements.txt
